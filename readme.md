## About this site

It is for someone who wants to share him or her special experience around cooking by using some recipe-books. Also it was made of PHP/Laravel, and Laradock/Docker was used during its development.

## Dependency

- Programming Language: PHP 7.3.8
- Web Framework: Laravel 5.5, Bootstrap 3.3.5
- Javascript Framework: Vue.js 2.6.10
- Web Server: Nginx
- Development Environment: Docker (using Laradock https://github.com/Laradock/laradock.git)
